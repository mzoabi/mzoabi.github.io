Privacy Policy for Feature Flags Manager

Feature Flags Manager does not collect, sell, share, or transmit personal data.

The extension reads the URL of the current active browser tab only when the user opens the popup. This is used to display and edit query-string feature flags.

The extension may store favorite feature flag names and values locally using browser storage. This data stays on the user's device and is not sent to any external server.

The extension does not use analytics, tracking, advertising, remote code, or third-party data services.
